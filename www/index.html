An application for starting your web applications. Checkout from CVS and execute ant new -Dapp.name=yourApp -Ddb.name=database to instantly be up and running with a Tomcat/MySQL app. Uses Ant, XDoclet, Hibernate, Spring, JUnit, Cactus, StrutsTestCase, Canoo's WebTest, Struts Menu, Display Tag Library, JSTL and Struts (including Validator and Tiles). Features include CMA, Remember Me, Self Registration, Password Hint. The fuse to start your apps.
